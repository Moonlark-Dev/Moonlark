## 说明

Moonlark 是一个开放的、稳定的、非盈利的多功能聊天机器人。

Moonlark 是 Moonlark Project 的下一个项目，使用基于 Python3 的 Nonebot2 框架编写。

## 背景

如果你没听过 Moonlark，那你可能听过它的另一个名字——XDbot。

为了解决 XDbot2 代码过于混乱且兼容性极差的问题，我们决定开发一个新项目，它便是 Moonlark。

Moonlark 目前还在测试中。
