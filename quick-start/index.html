快速开始 ​

谁适合阅读本文档？ ​

在阅读本文档前，请确保您符合以下条件：

  1. 掌握基本的 Python3 语法，能够使用其进行开发。
  2. 了解 NoneBot2 框架的使用及其插件结构。
  3. 能够访问和使用 GitHub，了解 Git 的基本使用方法。
  4. 能够使用 nb-cli 和 poetry。

代码规范 ​

在仓库的 CONTRIBUTING.md 中，我们已地列出了需要遵守的代码规范，对于其中提到的由 Moonlark 实现的插件，我们在 插件表 中编写了文档。

TIP

有一部分编辑器或 IDE （如 PyCharm）可能在重构时在 src/ 和 src/plugins/ 下建立空的 __init__.py 文件，请删除它或添加为 Ignore。
