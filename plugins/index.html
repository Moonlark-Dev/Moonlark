插件表 ​

Moonlark 内置了一些 Library 型插件，以下是这些插件的说明及用法。

在开发到类似功能时，请考虑使用这些插件（避免重复造轮子）。

功能类 ​

  • LarkLang: 本地化插件
  • LarkUtils: 小型工具包
  • LarkUser: 用户数据
  • Render: Jinja 模板渲染器
  • LarkUID: API 鉴权相关
  • LarkHelp: 插件帮助
  • OpenAI: GPT 相关
  • Ranking: 排行榜制作工具

玩法类 ​

TIP

部分插件仅能操作已注册用户。

  • Email: Moonlark 内置邮件
  • Achievement: 成就相关
  • Bag: 用户背包
  • Item: 物品基类相关
  • Jrrp: 今日人品
